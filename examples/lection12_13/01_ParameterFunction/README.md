# Пример: Передача функций как параметров

## Описание

Данный пример демонстрирует **передачу функций как параметров** другим функциям в C++ - базовую концепцию функционального программирования.

## Суть примера

### Определение функций

Четыре математические операции:
- `addition(int x, int y)` - сложение
- `subtraction(int x, int y)` - вычитание
- `multiplication(int x, int y)` - умножение
- `division(int x, int y)` - деление

### Type alias для типа функции

```cpp
using FuncType = int(int, int);
```

Определяет тип функции, которая принимает два `int` и возвращает `int`.

### Функция, принимающая функцию как параметр

```cpp
void PassingFunc(FuncType fn, int x, int y) {
    std::cout << "Result = " << fn(x, y) << std::endl;
}
```

Функция `PassingFunc` принимает:
- Функцию `fn` типа `FuncType`
- Два целочисленных параметра `x` и `y`

## Принцип работы

1. **Выбор операции** - пользователь выбирает тип операции (1-4)
2. **Ввод данных** - пользователь вводит два числа
3. **Передача функции** - выбранная функция передается в `PassingFunc`
4. **Выполнение** - `PassingFunc` вызывает переданную функцию с параметрами

## Пример использования

```cpp
PassingFunc(addition, 5, 3);        // Result = 8
PassingFunc(subtraction, 5, 3);     // Result = 2
PassingFunc(multiplication, 5, 3);   // Result = 15
PassingFunc(division, 6, 3);         // Result = 2
```

## Преимущества

- **Гибкость** - одна функция может работать с разными операциями
- **Переиспользование кода** - логика вызова функции централизована
- **Расширяемость** - легко добавить новые операции
- **Абстракция** - `PassingFunc` не знает о конкретной операции

## Связь с другими темами

Этот пример подводит к изучению:
- **Лямбда-выражений** (`02_Lambda`) - более удобный способ передачи функций
- **Функторов** - объекты, которые можно вызывать как функции
- **std::function** - обертка для любых вызываемых объектов

## Альтернативные подходы

Вместо передачи указателя на функцию можно использовать:
- Лямбда-выражения: `PassingFunc([](int x, int y) { return x + y; }, a, b);`
- std::function: `void PassingFunc(std::function<int(int, int)> fn, int x, int y);`
- Функторы: объекты с перегруженным `operator()`
