# 08_Types - Работа с типами в шаблонах C++

## Описание примера

Этот пример демонстрирует **работу с типами как параметрами и результатами** в шаблонах C++. Показывает, как создавать type traits для проверки типов и преобразования типов на этапе компиляции.

## Ключевые концепции

### 1. Тип как параметр - Проверка типа
```cpp
template <class T>
struct is_int {
    static const bool value = false;
};

template <>
struct is_int<int> {
    static const bool value = true;
};
```
- **Базовый шаблон** - возвращает `false` для всех типов
- **Специализация** - возвращает `true` только для типа `int`
- **Type traits** - проверка свойств типов на этапе компиляции

### 2. Тип как результат - Удаление указателя
```cpp
template<class T>
struct remove_pointer {
    using type = T;
};

template<class T>
struct remove_pointer<T*> {
    using type = T;
};
```
- **Базовый шаблон** - возвращает сам тип для не-указателей
- **Специализация** - удаляет один уровень указателя
- **using type** - определение типа как результата

### 3. Практическое применение
- **decltype** - получение типа выражения
- **typeid** - получение информации о типе во время выполнения
- **Специализация шаблонов** - для разных типов указателей

## Особенности реализации

### Специализация шаблонов
- **Полная специализация** для конкретного типа (`int`)
- **Частичная специализация** для указателей (`T*`)
- **Базовые шаблоны** для общих случаев

### Работа с типами
- **decltype(expression)** - получение типа выражения
- **typename** - указание на то, что имя является типом
- **using type = T** - определение типа как результата

### Type traits
- **Проверка типов** на этапе компиляции
- **Преобразование типов** с помощью специализации
- **Метапрограммирование** с типами

## Ожидаемые результаты

```
=== ДЕМОНСТРАЦИЯ РАБОТЫ С ТИПАМИ В ШАБЛОНАХ ===

1. Проверка типа (тип как параметр):
   is_int<bool>::value = 0
   is_int<int>::value  = 1
   is_int<double>::value = 0
   is_int<char>::value = 0

2. Удаление указателя (тип как результат):
   Исходный тип: int*
   Тип после remove_pointer: int
   Значение переменной: 42

3. Работа с разными типами указателей:
   double* -> d = 3.14
   char* -> c = A
   int -> i = 200
```
